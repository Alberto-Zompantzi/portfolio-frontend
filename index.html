Sobre mí

Soy Alberto Zompantzi, Ingeniero de Software especializado en el desarrollo de interfaces web modernas y escalables con React.js. Con una base sólida en algoritmos y lógica de programación, me enfoco en crear aplicaciones de alto rendimiento utilizando Vite y JavaScript (ES6+). Cuento con capacidad demostrada gestionando el ciclo de vida completo de productos en producción, optimizando arquitecturas modulares y flujos de trabajo mediante GitFlow y CI/CD. Mi especialidad es transformar diseños complejos en experiencias fluidas bajo la metodología Mobile-First.

DivertiKids — Plataforma Web Interactiva (React)
https://divertikids.pages.dev/ | 2026 Problema Resuelto: Necesidad de una plataforma interactiva que gestionara contenido dinámico de eventos infantiles sin comprometer la velocidad de carga.

Solución Técnica: Arquitecturé una Single Page Application (SPA) escalable utilizando React.js, separando la lógica de negocio de la UI mediante componentes funcionales y hooks. Optimicé la gestión de contenido mediante el mapeo de datos estructurados en JSON, permitiendo actualizaciones de información ágiles sin necesidad de redeploy del código base. Implementé un pipeline de CI/CD con Cloudflare Pages, garantizando despliegues automatizados y una disponibilidad del 100%.

Casa Zompantzi — Solución Digital Para Empresa De Plásticos (React)
https://casazompantzi.pages.dev/ | 2025 Problema Resuelto: Baja presencia digital de un negocio familiar que requería una identidad profesional y optimización para motores de búsqueda para atraer clientes locales.

Solución Técnica: Desarrollé la interfaz utilizando React.js, priorizando el SEO técnico y el renderizado eficiente para mejorar el posicionamiento en buscadores. Implementé un sistema de diseño con CSS Custom Properties (Variables) para asegurar una consistencia de marca total y facilitar el mantenimiento visual. Apliqué la metodología Mobile-First, logrando una experiencia de usuario (UX) fluida y adaptada al comportamiento de navegación en smartphones.

Sunnyside Boutique — | E-commerce Landing & UI Showcase (JS Vanilla)
https://sunnysideboutique.pages.dev/ | 2025 Problema Resuelto: Falta de adaptabilidad visual en catálogos de productos complejos, lo que generaba altas tasas de rebote en dispositivos móviles.

Solución Técnica: Desarrollé una interfaz altamente responsiva utilizando técnicas avanzadas de CSS (Grid y Flexbox), resolviendo problemas de jerarquía visual en pantallas pequeñas. Estructuré el proyecto utilizando Vite para optimizar el bundle de activos (imágenes y estilos), reduciendo significativamente el Time to Interactive (TTI). Aseguré la accesibilidad del sitio mediante el uso de HTML5 Semántico, mejorando la legibilidad tanto para usuarios como para lectores de pantalla.
