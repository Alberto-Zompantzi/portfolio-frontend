# 🚀 Portfolio Profesional — Alberto Zompantzi

### [🌐 Demo en Vivo](https://albertozompantzi-portfolio.pages.dev/)

Esta es la versión refactorizada y modernizada de mi portafolio profesional. Migrada de Vanilla JS a una arquitectura robusta en **React + TypeScript**, centrada en el **rendimiento**, la **escalabilidad** y la **internacionalización**.

---

## 🛠️ Stack Tecnológico Moderno

### **Frontend (La Refactorización)**
- **React 18:** Arquitectura de UI basada en componentes para una mejor gestión de estados y reutilización.
- **TypeScript:** Tipado estático para un código más fiable y mantenible.
- **Vite:** Herramienta de frontend de próxima generación para una experiencia de desarrollo extremadamente rápida.
- **i18next:** Soporte completo de internacionalización (Inglés/Español).
- **Framer Motion:** Animaciones de alto rendimiento para una sensación premium.
- **Lucide React & Devicons:** Iconografía moderna y consistente con colores de marca específicos.
- **Variables CSS:** Sistema de diseño centralizado para una tematización fácil y consistencia visual.

### **Backend e Infraestructura**
- **Spring Boot 3:** API RESTful robusta para la gestión de leads y persistencia de mensajes.
- **PostgreSQL (Neon):** Base de datos relacional nativa de la nube.
- **Docker:** Backend contenedorizado para entornos de despliegue consistentes.
- **EmailJS:** Servicio integrado de entrega de correo electrónico.
- **Cloudflare Pages & Render:** Hosting de alta disponibilidad tanto para el frontend como para el backend.

---

## 🌟 Características Clave

- **Internacionalización Completa:** Cambio fluido entre inglés y español (ES-MX por defecto).
- **UI/UX Moderna:** Secciones de Proyectos y Hero completamente rediseñadas con iconos de marca coloreados y transiciones suaves.
- **Sistema de Contacto Híbrido:** Persistencia en tiempo real en una API personalizada de Spring Boot y notificaciones por correo instantáneas.
- **Rendimiento Optimizado:** Construido con Vite para un tamaño de paquete mínimo y activos optimizados.
- **Arquitectura Limpia:** Estructura de carpetas estrictamente organizada (`components`, `hooks`, `services`, `assets`, `i18n`).

---

## 📂 Estructura del Proyecto

```bash
src/
├── assets/     # Estilos globales y variables CSS
├── components/ # Componentes de UI reutilizables
├── i18n/       # Configuración de internacionalización
├── services/   # Integración de API y EmailJS
└── types/      # Interfaces de TypeScript
```

---

## 🚀 Inicio Rápido

1. Clona el repositorio
2. Instala las dependencias: `npm install`
3. Ejecuta el servidor de desarrollo: `npm run dev`
4. Construye para producción: `npm run build`
